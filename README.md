📅 Study Planner (스터디 플래너)
Study Planner는 하루를 계획하고 목표를 달성하기 위한 심플하고 강력한 웹 애플리케이션입니다.

복잡한 기능은 덜어내고, 학습자가 공부에 몰입할 수 있도록 돕는 핵심 기능들로 구성되어 있습니다.

✨ 주요 기능 (Key Features)
1. 📝 데일리 플랜 (Daily Plan)
    - 할 일 관리: 오늘 해야 할 일을 직관적으로 기록하고 관리합니다.

    - 카테고리 자동 분류: 입력된 내용을 분석하여 공부, 운동, 식사, 휴식 등으로 카테고리를 자동으로 태깅합니다.

    - 내일로 미루기: 오늘 못한 일은 화살표 버튼 하나로 내일 계획으로 손쉽게 이동할 수 있습니다.

2. 🎯 목표 모드 (Goal Mode)
    - D-Day 설정: 중요한 시험이나 목표 날짜를 설정하여 남은 기간을 시각적으로 확인합니다.

    - 몰입 환경: 목표 모드 활성화 시 집중력을 높여주는 다크 테마(Dark Mode)로 자동 전환됩니다.
    
    - 목표 관리: 목표 달성 여부를 기록하고, 진행 중인 목표를 체계적으로 관리합니다.

3. ⏱️ 스마트 타이머 (Smart Timer)
    - 공부 시간 측정: 일상 모드에서는 순수 공부 시간을 측정(Count Up)하여 기록합니다.

    - 목표 시간 도전: 목표 모드에서는 설정한 시간 동안 집중할 수 있도록 카운트 다운(Count Down) 타이머를 제공합니다.

4. 📊 성장 리포트 (Statistics)
    - 학습 통계: 주간(Weekly) 및 월간(Monthly) 총 학습 시간을 그래프로 시각화합니다.

    - 활동 분석: 파이 차트를 통해 공부, 운동, 휴식 등 나의 활동 비율을 한눈에 파악할 수 있습니다.

    - 목표 히스토리: 과거에 설정했던 목표들의 달성/실패 이력을 조회할 수 있습니다.

5. 🎵 부가 기능
    - 뮤직 플레이어: 공부하며 들을 수 있는 YouTube 음악을 배경 재생할 수 있습니다.

    - 강의 검색: 학습에 필요한 강의 영상을 바로 검색하고 시청할 수 있습니다.

    - 데이터 백업: 나의 학습 기록을 JSON 파일로 다운로드하여 안전하게 보관할 수 있습니다.

🛠️ 기술 스택 (Tech Stack)
<Frontend>
    Framework: React, Vite
    Styling: Tailwind CSS
    Chart: Recharts (데이터 시각화)
    Icons: Lucide React
    Network: Fetch API

<Backend>
    Runtime: Node.js
    Framework: Express.js
    Database: MySQL
    Security: Bcrypt (비밀번호 암호화), JWT (인증)
